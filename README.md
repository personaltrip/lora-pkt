A LoRa Packet Parser only by Html and JavaScript, static Web page.

Generated by Gemini.

Packed and published on Docker hub.

Use Docker Compose to deploy`sudo docker compose up -d`.
```
services:
  lora-pkt:
    image: steammilk/lora-pkt:v5.1
    container_name: lora-pkt
    ports:
      - "8088:80"
    restart: always
```
